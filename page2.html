Solving Absolute Value Functions

Use graphs and equations to evaluate f(a) and solve f(x) = b — two of the most essential skills in function notation.

📍 What does f(a) mean?

f(a) means evaluate the function at x = a — you substitute a specific number in for x and compute the output.

On a graph, you can find f(a) visually:

  1. Start at x = a on the x-axis.
  2. Move vertically (up or down) until you reach the graph.
  3. Move horizontally to the y-axis and read the value.
  4. That y-value is f(a).
Example: If g(x) = |x| + 1, then g(3) = |3| + 1 = 4. The point (3, 4) is on the graph.

🎯 What does f(x) = b mean?

f(x) = b means find all x-values where the output equals b — you are solving for the input(s) that produce a specific output.

On a graph, you can find solutions visually:

  1. Draw a horizontal line at y = b.
  2. Find every point where that line crosses the graph.
  3. Drop vertically from each crossing to the x-axis.
  4. Each x-value you land on is a solution.
Key insight: An absolute value function can have 0, 1, or 2 solutions depending on where the horizontal line falls relative to the vertex.
