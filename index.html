Absolute Value Function Transformations

Use the sliders to change each parameter and watch the graph respond in real time.

📖 Background

The parent function for absolute value is f(x) = |x|. It creates a V-shaped graph with its vertex at the origin (0, 0) and both arms rising at a slope of 1. Any transformation of this function can be written in vertex form:

g(x) = a|x − h| + k

Each letter — a, h, and k — is a parameter that controls a specific transformation. Adjusting a parameter changes the shape or position of the graph while keeping it a V-shape. The vertex of g(x) is always located at the point (h, k).

a — Vertical Stretch / Compression

Controls how steep or wide the V is. When |a| > 1 the graph is narrower (stretch). When 0 < |a| < 1 the graph is wider (compression). A negative value of a flips the V upside down — a reflection over the x-axis.

h — Horizontal Translation

Moves the vertex left or right. When h > 0 the graph shifts right h units. When h < 0 it shifts left. The equation reads (x − h), so the shift direction can feel counterintuitive — pay attention to the sign!

k — Vertical Translation

Moves the vertex up or down. When k > 0 the graph shifts up k units. When k < 0 it shifts down. This is the same as adding or subtracting a constant from every output value of the function.

🔬 Guided Exploration Activities

Activity 1

Vertical Stretch & Compression

  • Reset all sliders, then drag a slowly from 1 to 3. How does the V change?
  • Now drag a from 1 toward 0. Describe what happens to the width.
  • Set a = −1. What does the negative sign do to the graph?
  • When you only change a, where does the vertex stay? Why?
Activity 2

Horizontal Translations

  • Reset, then drag h to a positive value. Which direction does the vertex move?
  • Try a negative value of h. Is the direction what you expected?
  • The equation reads |x − h|. When h = 3, write the full equation and explain why the graph moves right.
  • What are the coordinates of the vertex when only h changes?
Activity 3

Vertical Translations

  • Reset, then drag k upward. What changes — and what stays the same?
  • Set k = −4. Where is the vertex now?
  • Does changing k ever affect the steepness of the V?
  • What is the connection between the value of k and the vertex's y-coordinate?
Activity 4

Combining All Transformations

  • Set a = 2, h = 3, k = −1. Write the equation shown for g(x).
  • Before adjusting, predict where the vertex will be. Then check your prediction.
  • Challenge: Without using Reset, move the sliders back so g(x) matches f(x) exactly.
  • Find settings that create a wide, upside-down V with vertex at (−2, 4).
