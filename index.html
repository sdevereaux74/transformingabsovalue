Absolute Value Function Transformations

Use the sliders to change each parameter and watch the graph respond in real time.

📖 Background

The parent function for absolute value is f(x) = |x|. It creates a V-shaped graph with its vertex at the origin (0, 0) and both arms rising at a slope of 1. Any transformation of this function can be written in vertex form:

g(x) = a|x − h| + k

Each letter — a, h, and k — is a parameter that controls a specific transformation. Adjusting a parameter changes the shape or position of the graph while keeping it a V-shape. The vertex of g(x) is always located at the point (h, k).

a — Vertical Stretch / Compression
  • Controls how steep or wide the V-shape is
  • When |a| > 1, the graph is narrower (vertical stretch)
  • When 0 < |a| < 1, the graph is wider (vertical compression)
  • When a is negative, the V flips upside down — a reflection over the x-axis
h — Horizontal Translation
  • Moves the vertex left or right along the x-axis
  • When h > 0, the graph shifts right h units
  • When h < 0, the graph shifts left |h| units
  • The equation reads (x − h), so the direction can feel counterintuitive — pay attention to the sign!
k — Vertical Translation
  • Moves the vertex up or down along the y-axis
  • When k > 0, the graph shifts up k units
  • When k < 0, the graph shifts down |k| units
  • The value of k is always the y-coordinate of the vertex
